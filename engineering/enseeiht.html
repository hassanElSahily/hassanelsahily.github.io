ENSEEIHT — Coursework & Engineering Projects

As part of my studies at ENSEEIHT, I am engaged in an engineering curriculum focused on computer systems, architecture, and networks. This training emphasizes both theoretical foundations and hands-on projects addressing low-level systems, communication infrastructures, and performance constraints.

Computer Architecture & Digital Design (Archi) - On going

The Archi project focused on the design and implementation of a simplified UART (Universal Asynchronous Receiver/Transmitter) using VHDL, targeting a hardware-level understanding of serial communication interfaces.

  • Design of a modular UART architecture (TX, RX, clock, and control units).
  • Implementation of asynchronous serial transmission (RS232 format).
  • Clock division and timing control for transmission and reception.
  • Management of control registers, interrupts, and transmission errors.

This project provides hands-on experience with hardware description languages, finite state machines, and the interaction between hardware components and processor-level control.

Interconnection of Autonomous Systems (Interco) - On going

The Interco project focuses on the design and deployment of an autonomous system (AS) providing network services to both enterprise and residential clients, as well as its interconnection with other autonomous systems.

  • Design and implementation of an autonomous system using dynamic routing (OSPF).
  • IP addressing plan based on an assigned address block and hierarchical allocation to clients.
  • Deployment of enterprise networks distributed over two sites connected through different AS.
  • Configuration of secure inter-site communication using VPN mechanisms.
  • Initial implementation of Quality of Service (QoS) for enterprise traffic (work in progress).

This project provides a concrete understanding of large-scale network design, routing protocols, and the interaction between service providers, enterprise networks, and end users.

Compilation & Language Processing - On going

This project focused on the design and implementation of a compilation chain for a domain-specific language, emphasizing the transformation from high-level code to low-level executable representations.

  • Lexical and syntactic analysis using formal grammars.
  • Construction and traversal of abstract syntax trees (AST).
  • Semantic checks and symbol table management.
  • Code generation targeting a low-level or intermediate representation.

This project strengthened my understanding of compiler architecture, language semantics, and the close relationship between programming languages and system-level execution.

Academic Perspective

The ENSEEIHT curriculum plays a central role in shaping my long-term academic objectives. Through these projects, I developed a strong interest in systems research, supported by the diversity of courses ranging from Internet architecture and networking fundamentals to local area networks, telecommunications, and operating system architecture.

This experience directly supports my ambition to pursue a PhD and contribute to research and teaching in computer science, with a focus on rigorous system design and knowledge transmission.
